### Ruby

```ruby
# This is a single-line comment

=begin
This is a multi-line comment
that spans multiple lines
=end

class Person
  attr_reader :name

  def initialize(name)
    @name = name
  end

  def greet
    puts "Hello, #{@name}!"
  end
end

person = Person.new('John')
person.greet
```

### Python

# This is a single-line comment

"""

    def greet(self):

person = Person('John')
